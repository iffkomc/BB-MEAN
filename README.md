# BB-MEAN
Photobattles project

### What's under the hood?
  * Express4
  * Angular 1.5
  * MongoDB
  * NodeJS
  * Gulp
  * Jade
  * Sass
  * BEM
  * REST API
  * Image Magic / Graphics Magic (Uploading photos)

### To install:
1. Install global node modules (express, image-magic, gulp, etc.)
2. Install local node modules `npm install`
3. Run mongo `mongod`
4. `cd ./lib`
5. `gulp`
6. Go to ./server
7. Run `node app`
